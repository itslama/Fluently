# Fluently
 UI/UX design project developed for Human Computer Interaction, proposing FLUENTLY, a mobile English learning app aimed at adult and young learners. The design addresses common pain points in existing language apps — childish interfaces, long lessons, and limited speaking practice — by offering a modern scroll-based experience, an AI conversation agent, and optional live sessions with human tutors.
The project covers the full HCI design process, from user research to a tested interactive prototype.
Key Features (Proposed): Scroll-based short lessons · AI speaking practice · Live tutor booking · Kids mode · Progress dashboard · Vocabulary quizzes · Heuristic & usability evaluation
Deliverables: User research (interviews + questionnaire) · Requirements · Personas · Figma prototype · Heuristic evaluation · Usability testing.
